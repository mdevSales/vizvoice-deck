Why nothing has fixed this

Every "accessibility fix" assumes the goal is reading the chart.

Alt-text bandaids

Static one-line descriptions, written once by a sighted analyst. The data changes. The alt text doesn't.

Screen reader hell

Tableau renders as unlabeled SVG. VoiceOver announces "graphic, graphic, graphic" and gives up.

Tabular fallbacks

Data tables buried three clicks deep. Better than nothing. Worse than useful.

The reframe

What if you didn't
need to see the chart

The insight was never in the pixels. It was in the answer to the question you'd ask about them.

How it works

Ask the dashboard out loud.
Hear the answer back.

Grounded, not guessed

The agent reasons over the same semantic model as every chart in the org.

No RAG. No custom index. No drift between what a sighted user sees and what a blind user hears — one Data Cloud definition, honored by both.
